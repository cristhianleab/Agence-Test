Dependencias de node:
	-Express
	-Body parser
	-Mysql2
	-Cors
	-Body-parser

Para ejecutar el proyecto se debe correr localmente app.js de la carpeta backend mediante node app.js.
La conexión a la base de datos en mysql tiene el usuario 'root' y el password '' (default).

Luego, ejecutar el front abriendo el archivo index.html.
